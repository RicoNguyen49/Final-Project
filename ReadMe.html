Game: Tic-Tac-Toe

Description:

Tic Tac Toe is a classic two-player game typically played on a 3x3 grid. Players take turns marking spaces with their respective symbols "X" and "O", with the objective of getting three of their symbols in a row, either horizontally, vertically, or diagonally.

Game: Space Exploration

Description:

Space exploration takes inspiration from flappy bird the classic space invaders arcade game but without the invaders. The goal of the game is to achieve the highest score, traveling through different levels as it gets faster and obstacles change at each milestone. The user will control the spaceship with the arrow keys and have to advoid hitting the lazer obstacles as it traverse the universe. The lazers will change colors randomly with most of them being bright and easy to see. the game includes a black out stage where the lazers will turn black making it harder for the user to advoid the obstacle.

Game: Retro Match

Description:

This game is a memory matching game that challenges players to match pairs of Pixel Art. The table of images will be randomly shuffled every round. each two cards fliped will count as a attempt but when you get it correct it will not count as an attempt. The goal is to get the lowest amount of attempts to match all the pixel images.
